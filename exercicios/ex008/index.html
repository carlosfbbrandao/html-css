Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito, usando FORMA, para isso foi usado a tag <b> (não semântica).

Nesta frase, temos um termo em destaque, usando DESTAQUE, para isso foi usado a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <I> (não semântico).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag ≤mark> cria uma marcação em HTML de padão amarelo.

Neste exemplo 01 vamos alterar apenas um elemento do texto marcado dentro da tag <mark> que será a cor da marcação, colocando um parâmentro "style", que não é HTML e sim CSS.

Neste exemplo 02 vamos alterar de dois elementos do texto marcado dentro da tag <mark> que será a cor da marcação e a cor do texto, colocando um parâmentro "style", que não é HTML e sim CSS.

Neste exemplo 02

Uma forma de padronisar todos os textos marcados para uma mesma cor temos que usar a tag <style> em sua forma padrão (sem o parâmetro CSS Style), e no cabeçalho colocar uma tag <style> e dentro dela crio um seletor para "mark" que será "backgroud-color" e ecolho a cor que todos os marks em sua forma padão irá assumir.

OBS: as tags <mark> que tiverem com seu valor padrão (sem o parâmetro CSS "style") sofrerão alteração de acordo com a tag <mark> colocada no cabeçalho. As tags <mark> que possuirem apenas um dos elementos alterados fora do padão, o outro sofrerá alteração de acordo com o determinado no cabeçalho (conforme exemplo 01). Texto grande e Qequeno

Estamos criando um texo grande e um texto pequeno nesse parágrafo. Para o texto grande usaremos a tag <big> (nao semântica) e para o texto pequeno usaremos a tag <small> (semântica).

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado, usando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag <ins>.

Texto Sobrescrito

Para inserir coisas do tipo X20+3, utilizamos a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O, utilizamos a tag <sub>.
