Outras formatações

Código-Fonte / Pré-formatação

Para deixar uma fonte mono-espaçada (cada caracter ocupando o meso espaço) utilizamos a tag <code>. veja no exemplo abaixo:
document.getElementById (sem a tag <code>)
document.getElementById (com a tag <code>)

Para criar um código com identação (igual ao que é digitado) temos que utilizar a tag <pre>. veja o exemplo abaixo:

Com a tag <code> mas sem a tag <pre>:

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é par) else: print(f'O número {num} é impar') print('Fim do programa')

Com a tag <code> e com a tag <pre>:

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é par)
    else:
        print(f'O número {num} é impar')
    print('Fim do programa')

Citações Simples

Para citações simples a forma correta de utilizar as "" é utilizando a tag <q> que faz com que sejam colocadas com um sentido de citação, e não apenas aspas, como por exemplo:

como diria o pai de um amigo: o computador é um burro muito rápido.

Citações Completas

Para citações mais longas utilizamos a tag <blockquote>, como por exemplo:

Como diria Jaff Noble, no seu livro HTML para Leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML neste capitulo descrevem os blocos de texto.

E dentro da tag <blockquote> podemos colocar um parâmetro "cite" e nele colocar a URL do altor.

Abreviações

Existe uma maneira de colocar uma explicação para uma abreviação utilizando a tag <abbr>, como no exemplo abaixo:

Estou estudando HTML e CSS. Estou adorando!

Parando o mouse encima das siglas, ele exibe os textos que foram colocados na tag <abbt>.

Texto Invertido

Para fazer com que um texto apareça no navegador de forma invertida, utilizamos a tag <bdo>. Veja no exemplo abaixo:

Estou aprendendo a criar coisas em HTML. (na forma normal).

Estou aprendendo a criar coisas em HTML. (na forma invertida).
