Trabalhando com Listas

Listas Ordenadas

As listas ordenadas são aquelas em que a ordem faz diferença. Para facilitar a ordenação das listas ordenadas, utilizamos a tag <ol> (order list) com isso todos os itens colocados na lista recebem sua 'numeração' automaticamente, e assim, se precisar acrescentar algum item em algum lugar na lista, suar ordem é ajustada automaticamente. Cada item da lista tem que entrar com a tag <li> (list iten). Veja o exemplo abaixo:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

A lista vem automaticamente ordenada com a forma numérica, mas também podemos utilizar outras formas para ordenar utilizando o parâmetro "type" dentro da tag <ol>, que dará mais 5 opções de formas de ordenação (1, A, a, I, i).

Veja os exemplos de formas de ordenação utilizando as opções dentro do parâmetro "type":

Parâmetro "1" (numérico):

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Parâmetro "A" (alfabético maiúsculo):

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Parâmetro "a" (alfabético minusculo):

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Parâmetro "I" (romano maiúsculo):

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Parâmetro "a" (romano minusculo):

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Podemos também selecinar de onde a ordenação irá começar utilizando também o parâmetro "start" e informando de qual posição irá iniciar.

Listas não Ordenadas

As listas não ordenadas são aquelas em que a ordem não faz diferença. Para estas listas vamos utilizar a tag <ul> (unorder list). Veja o exemplo abaixo:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Como padrão a marcação das listas vêem com uma "bolinha" que é a tipo "disc", mas temos algumas opções que podemos alterar utilizando o parâmentro "type" dentro da tag <ul> e podemos colocar as opções "disc" (que é a padrão), "circle", "square". Vejá os exemplos abaixo:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Lista de definição

Criamos a lista de descrição utilizando a tag <dl> (description list), que é uma lista onde colocamos os itens de acordo com o que ele é. Para um item que seja um "termo" utilizamos a tag <dt>, e para um item que seja uma "descrição" utilizamos a tag <dd>. Veja no exemplo abaixo:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a acriação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
